# medRxiv Submission Checklist

Complete these author-controlled items before posting.

## Required Before Submission

- Public repository URL for adapted code, saved outputs, analysis tables, figures, and manuscript-generation scripts.
- Permanent archive DOI if desired, for example through Zenodo, OSF, or another repository.
- Corresponding-author email.
- Confirm final author list and affiliations: current manuscript lists Koyar Afrasyab, M.D.; Kinvectum AB.
- Confirm funding statement: current manuscript states the project was funded by Kinvectum AB.
- Confirm competing-interest statement: current manuscript states that Koyar Afrasyab, M.D. is the founder of Kinvectum AB, which funded the project.
- Author-contribution statement.
- Confirmation that no API keys, `.env` files, or credential-containing notes are included in the public release.
- Confirmation that restricted JAMA/NEJM/proprietary image assets are not redistributed.

## Suggested Data Availability Wording After Public Release

Replace the Data and Code Availability paragraph with:

> The adapted evaluation code, saved model outputs, analysis CSVs, figures, and manuscript-generation scripts are available at [PUBLIC_REPOSITORY_URL]. An archived version is available at [DOI, if assigned]. Public dataset use follows the terms of the original dataset sources. Restricted JAMA, NEJM, proprietary unseen-set, and visual-substitution source images are not included.

## Suggested Author-Contribution Wording

Use only if accurate:

> Koyar Afrasyab conceived the study, configured the updated evaluation, supervised model runs, interpreted the results, and drafted and revised the manuscript.
